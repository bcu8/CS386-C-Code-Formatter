About This Project

Purpose:

The purpose of this project is to make it easier to implement best practices into your code. Not all the GUR violations are guarded against. Below you can find a list of the GUR violations checked by the latest version of our site.

How to use:

Write your code and ensure it is free of compiling errors and warnings. Upload the file via the account page. Once uploaded you may view our suggestions and use our white space formatter. If the white space formatter is run, our saved copy of your file is replaced with the formatted version and you may click "download file" to download the white space formatted version.

The white space formatter will:

  • replace tabs with 4 spaces
  • automatically align curly braces (as long as they are on a line of their own)
  • Add consistent indentation to blocks of code

The following items are known to cause erroneous suggestions:

  • Use of curly braces within strings
  • Code that has compiling errors
