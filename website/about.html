About This Project

Purpose

The purpose of this project is to make it easier to implement best practices into your code. The code formatter will NOT alter the file(s) you provide. Instead a new formatted file will be created for download onto your device. Not all the GUR violations are guarded against. Below you can find a list of the GUR violations checked by the formatter.
